# wilderness-reflections

This repository contains a page-verified Markdown transcription of the supplied scan of *Wilderness Reflections*, a book distributed to Philmont staff from the 1980s to the 2000s.

All 76 leaves present in the PDF have been processed. The supplied scan omits original Author Index pages 82-84 and First Line Index pages 86-90; placeholders and the source-gap details are documented in the manuscript and editorial notes pending replacement scans.
